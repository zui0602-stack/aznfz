# AI 使用本知識庫的規則

## 優先順序

1. 先讀任務需求。
2. 找 `all_tutorials_table.md` 中相關分類。
3. 讀對應頁面筆記。
4. 若要精確函數參數，再回官方文檔搜尋原文。
5. 寫腳本前先列「使用模組」。
6. 寫腳本時每一步都要有成功判斷、失敗處理、超時、日誌。

## 模組選擇順序

1. 無障礙控件：能讀到 text/id/className/tag 時優先。
2. OCR：文字按鈕、文字狀態、遊戲文字。
3. 找圖：固定圖示、固定按鈕、遊戲 UI 圖示。
4. 找色/多點找色：紅點、狀態色、固定色塊。
5. YoloV8：畫面複雜、普通識別不穩時才用。
6. ADB/HID/Root：普通點擊不穩或環境需要時才用。

## 新手禁忌

- 不要一開始做完整超大腳本。
- 不要只靠座標和延時。
- 不要全屏 OCR 連續掃。
- 不要滑動中 OCR。
- 不要把 token、卡密、帳密放 GitHub。
- 不要在沒有成功判斷的情況下連續點擊。
