# 萬龍覺醒專案規則

## 目前核心目標

優先做單角色採集，之後才做多角色、每日任務、部隊檢查、倒數總控。

## 單角色採集流程

```text
主城
→ 偵測出城
→ 點出城
→ 點放大鏡
→ 選資源
→ 選等級
→ 搜尋
→ 點資源點
→ 點採集
→ 判斷創建部隊
   ├─ 有創建部隊：點創建部隊 → 點行軍 → 下一輪
   └─ 無創建部隊：停止派遣 → 進入部隊檢查
```

## 停止條件

不要用 5/5 當主要停止條件。主要判斷是：點「採集」後是否出現「創建部隊」。

## 識別方式

- 出城：找圖/OCR
- 放大鏡：找圖
- 資源類型：找圖/座標
- 等級：OCR/座標
- 採集：OCR
- 創建部隊：OCR
- 行軍：OCR/找圖
- 紅點：找色
- 部隊狀態：OCR

## 滑動規則

滑動時不偵測；滑動停止後延遲 500~1000ms 再 OCR/找圖。

## 時間規則

使用 24 小時制。若開始時間大於結束時間，表示跨日，例如 18:50 到隔天 07:59。
